= Heroku SQL

Plugin that provides a sql console for your heroku app.

== Installation

To install:

    $ heroku plugins:install git://github.com/ddollar/heroku-sql-console.git

== Usage

    $ heroku sql
    SQL console for example.heroku.com
    SQL> SELECT * FROM users;
